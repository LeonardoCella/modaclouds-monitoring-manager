#User Manual

## Installation

Requirements:
* Java 7 JDK
* Apache Maven 3+
* git

```
git clone https://github.com/deib-polimi/modaclouds-monitoring-manager.git
cd modaclouds-monitoring-manager
git checkout tags/v1.1.1
mvn package assembly:single
cd ..
```
