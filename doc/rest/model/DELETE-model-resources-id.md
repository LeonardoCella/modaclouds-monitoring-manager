# Metrics

	DELETE /model/resources/:id

## Description
Delete the specified resource from the knowledge base

***

## URL Parameters

None.

***

## Response

**Status:** **204 No Content**

***

## Errors

* **404 Resource not found** - The resource does not exist

***

## Example
**Request**

	DELETE v1/model/resources/vm1
